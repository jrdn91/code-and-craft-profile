# Code and Craft Curse Force Profile

## Instal Instructions

- Click the green "Code" button
- Click "Download ZIP"
- Open CurseForge
- Navigate to Minecraft
- Click "Create Custom Profile" at the top
- Under the title, click the "import" link
- Navigate to the downloaded ZIP
- Launch minecraft
